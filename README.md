## Overview:
A small responsive website built as part of a software engineering bootcamp to practice core front-end development skills.
The site highlights Sacramento-area running events and includes multi-page navigation, event detail pages, and form handling through a third-party service.
It demonstrates HTML, CSS, JavaScript, and static site deployment using GitHub Pages.

## Repository:
https://github.com/needhamjoshua/SEB-Project_1.git

## Live Site:
https://needhamjoshua.github.io/SEB-Project_1/index.html

## Features
- Multi-page responsive layout
- Event browsing by location or distance
- Individual event detail pages
- Registration and contact forms with text validation
- Form submissions handled via Formspree
- Static hosting with GitHub Pages

## Technologies Used:
- HTML5
- CSS (Bootstrap + Bootswatch Vapor theme + custom styles)
- JavaScript
- Formspree
- GitHub Pages

## Notes
Originally created as coursework for a software engineering bootcamp and expanded as a practical front-end exercise.
